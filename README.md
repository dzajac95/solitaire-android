# Solitaire Android App

A simple app for playing Klondike soliataire on Android.
This is a personal project to explore some basic game development, and to provide for myself an ad-free way of playing solitaire on my phone :)

Project structure was started from the [raymob](https://github.com/Bigfoot71/raymob) template for creating Android apps using [raylib](https://github.com/raysan5/raylib).

## Setup

For building, I've replaced the standard Gradle junk with [nob.h](https://github.com/tsoding/nob.h).

You will need to have downloaded and installed the android [NDK](https://developer.android.com/ndk/downloads) and [SDK](https://developer.android.com/tools/sdkmanager).

From there, set the following environment variables to point the build at your install locations:
- `ANDROID_NDK_ROOT` <-- NDK installation root
- `ANDROID_HOME` <-- SDK installation root, following the structure generated by `sdkmanager`
- `ANDROID_BUILD_TOOLS` <-- specific build-tools installation under ANDROID_HOME/build-tools

Optionally, set `JAVA_HOME` to use a specific java installation.

Now just run:
```sh
gcc -o nob nob.c
./nob
```
and you should get a signed APK under `./build/`.

If you have an android device attached, you can install the APK with `./nob install`.
